# Demo Package

This folder contains the files needed for a quick project demo.

## Contents

- `demo_checklist.md`: quick checklist for what to show during the demo.
- `demo_script.md`: short speaking script for the demo.
- `key_screenshots/`: layout, SPICE, DRC/LVS/timing proof screenshots.
- `key_plots/`: final plots used in the report/presentation.
- `final_outputs/`: final cleaned PDF and supporting zip archives.

This demo package is meant to be a lightweight presentation folder, not the full project repository.
